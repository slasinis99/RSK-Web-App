RSK / Two-Rowed Array / Matrix Explorer

Enter semistandard tableaux, a two-rowed array, or a matrix. The app converts between them and shows the intermediate steps.

Input

Choose one representation and click convert.

One row per line, entries separated by spaces.
Must have the same shape as P.
The app can sort the array into lexicographic order automatically.
Nonnegative integer matrix. Rows correspond to top-row values, columns to bottom-row values.

Output

The outputs below are synchronized: tableaux, two-rowed array, and matrix.
